## Installation

First, download the dependencies using Npm:

```shell
$ npm install
```

Then, run the project:

```shell
$ npm run serve
```

## Config

Copy the `.env.example` to `.env.[ENV]`:


```shell
# local
cp .env.example .env.local

# production
cp .env.example .env.production
```
